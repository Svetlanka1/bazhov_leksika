# Пояснительная записка, или "Как пользоваться продуктом?"
Здесь можно скачать и посмотреть только код и картинки, используемые в прототипе.
Посмотреть, как работает прототип в деле, мы предлагаем без скачивания - с помощью черновика Яндекс.Игры (https://yandex.ru/games/app/483291?draft=true&lang=ru)
Исходя из этого и будет описана дальнейшая инструкция.

### Инструкция к запуску:
- Перейти по ссылке на Яндекс.Игры (см. выше)
- Дождаться загрузки
- Нажать “Начать”
- Пользуясь левой кнопкой мыши выбрать сказ (в данной версии функциональна только “Малахитовая шкатулка”)
- Для прохождения новеллы  используем ЛКМ / пробелом / Enter

### Подсказки (ответы для мини-игр):
- причтелось
- бассенька 
- затравка

### Описание ограничений:
- Контент. В текущей версии представлен только один короткий сюжетный эпизод (на основе сказа «Малахитовая шкатулка») для демонстрации ключевых механик, нарратив будет дорабатываться и расширяться. Сценарии остальных сказов находятся на стадии ранней разработки.
- Графика. Используются временные ассеты, найденные на бесплатных ресурсах или сгенерированные ИИ. Финальный художественный стиль находится в разработке. 
- Анимации и звук отсутствуют. Приоритетом была проверка игровых механик и логики. В дальнейшем звуки будут добавлены в игру. Анимации, возможно, тоже, но это зависит от художника, которого мы найдем.
- Игровые задания (мини-игры) реализованы в упрощённом, черновом виде для проверки логики.
- Задание “Пяльце”: В текущей версии присутствует только схематичное изображение и одно слово (верное). В финальной версии игроку предстоит найти «дивное» слово по его определению среди вышитого узора (возможно, будет несколько итераций данного задания).
- Задание “Малахитовая стена”: Вместо стилизованных букв-самоцветов пока используются обычные разноцветные буквы (сейчас правильные буквы, из которых нужно складывать слово - чёрные). В дальнейшем это будет исправлено.
- Сценарий использования. Прототип ориентирован на использование в классе под руководством учителя (в качестве обращений в игре используются “вы”, “ребята”). Поэтому для  самостоятельного домашнего прохождения в текущем виде игра не подходит.

### Список багов/рисков:
- В зависимости от скорости интернет-соединения возможна задержка при загрузке спрайтов персонажей (временное отображение в низком разрешении - неких пиксельных пятнах).
- Из-за того, что данный прототип разработан исключительно по сказу “Малахитовая шкатулка”, но в будущем планируется 3 главы (по разным сказам). То сейчас в начале игры присутствует “выбор без выбора”: какой бы сказ вы не выбрали, вас отправят на прохождение главы по “Малахитовой шкатулке”. В дальнейшем будет настоящая развилка.
- Прототип оптимизирован для просмотра с десктопных устройств. Работа на мобильных устройствах не гарантируется и не является целевым сценарием.
