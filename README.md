# bazhov_leksika
Визуальная новелла (веб-формат) по сказам Бажова для изучения/закрепления сложной лексики из текстов.
Что это?
«Бажов в поиске слов» — это прототип образовательной визуальной новеллы, созданный для уроков литературы в 5-6 классах. Игра помогает учителю превратить знакомство со сложными сказами Павла Бажова в увлекательное приключение для всего класса.

В данном репозитории представлен главный код новеллы (файл script_novel).

Проект разрабатывается в рамках гильдии «Бажов Цифровой» для платформы «Словарь Бажова».

Зачем это нужно?
Язык сказов Бажова полон устаревших слов и реалий, которые становятся барьером для современных школьников. Учителям не хватает качественных цифровых материалов для работы на уроке.

Наш проект решает эти проблемы:

- Для учеников: Сложные слова объясняются визуально и в контексте сюжета, а не через сухой словарь. Игровой формат повышает вовлечённость.
- Для учителя: Это готовый инструмент для урока, который экономит время на подготовку и гарантированно удерживает внимание класса.
- Для платформы «Словарь Бажова»: Игра становится увлекательным «проводником», который на практике показывает ценность основного ресурса.

Как это работает? (Основные механики)

- Короткие сюжетные эпизоды (5-10 минут), которые учитель запускает на проекторе или интерактивной доске.
- Коллективное принятие решений: Класс обсуждает и выбирает варианты ответов за главного героя.
- Визуальное объяснение слов: Значение устаревших слов раскрывается через контекст диалогов, стилизованные подсказки и иллюстрации.
- Мини-игры для класса: В конце эпизода — групповые задания («найди слово», «посчитай украшения») для закрепления материала.

Кто создаёт проект?
Команда гильдии «Бажов Цифровой» университета ИТМО в рамках написания выпускной квалификационной работы.
